# Ping App 1.0.0
This app for monitoring domains and servers. 
This app contains next funcionality:

- Add IP and Domans.
- Save data to file
- Get information about current status 
- Run and Pause

In this project used Thread to run Ping sygnals and updated DataGrid in realtime.
